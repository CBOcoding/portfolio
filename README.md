# Claudio Borromei | Software Engineer Portfolio

A modern, responsive portfolio website showcasing my journey in Software Engineering. Built with a focus on performance, accessibility, and cross-language support.

[View Live Demo](https://claudioborromei.vercel.app/)

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox, CSS Variables)
- **Interactivity:** Vanilla JavaScript
- **Forms:** Integrated with Web3Forms API
- **Design:** Glassmorphism influences, Dark/Light Mode support

## ✨ Key Features
- **Bi-directional Language Support:** Instant toggle between English and Italian without page reloads.
- **Dynamic Theming:** Smooth transition between Light and Dark modes using CSS Variables and LocalStorage.
- **Responsive Layout:** Mobile-first approach with horizontal project scrolling.
- **Clean Documentation:** Code follows a strict commenting standard to assist with maintainability and peer review.

## 📂 Project Structure
├── assets/             # Static images and headshots
├── index.html          # Core structure & SEO metadata
├── style.css           # Theme variables & responsive design
└── script.js           # Theme, Language, and Scroll logic
